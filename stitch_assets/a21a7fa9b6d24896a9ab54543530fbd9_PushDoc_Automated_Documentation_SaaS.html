Curated Questions

How do we handle webhooks? expand_more

PushDoc integrates directly with GitHub Webhooks. When you push code, GitHub notifies our analyzer which immediately pulls the diff, parses your AST, and generates updated documentation.

What about code privacy? expand_more

We never store your source code. Our analyzer works on an ephemeral instance, extracts documentation metadata, and immediately purges the local copy.

Can I customize the templates? expand_more

Yes! Premium users can upload their own liquid-based documentation templates to match their brand guidelines exactly.
